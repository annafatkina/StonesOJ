# StonesOJ
Project with 3D stones

## Usage

###point coud generator

```cd pr/build ```

```cmake ../ && make```

```./point_cloud_stone <img1> <img2> <scale num>```


output is outpu0.xyz

###mesh generator

```cd mesh-saver/bould```
```cmake ../ && make```
```./mesh-saver -i <input  xyz> -o <output obj>```
